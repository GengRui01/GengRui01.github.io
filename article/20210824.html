Swing 概述

2021年08月24日
Java
留言

图形用户界面(GUI)

GUI 是指给用户操作的图形界面，也可以称为桌面程序

比如我们常常使用的 QQ 就是非常典型的桌面程序

甚至任何一种浏览器都是桌面程序,虽然浏览器里面展示的是 html 页面,但浏览这些 web 页面的浏览器是不折不扣的桌面程序

除此之外任何客户端程序也是图形用户界面程序,比如我们常玩的端游,游戏就是最复杂的桌面程序之一了

Swing 特点

Java 语言编写桌面程序用到的组件有两种,一种是 AWT , AWT 组件的Java应用程序在不同操作系统上执行时界面的显示是不同的

另一种就是 Swing , Swing 组件是新一代的图形界面工具, 使用 Swing 来开发图形界面比 AWT 更加优秀

因为 Swing 是一种轻量级组件，它采用纯 Java 实现，不再依赖于本地平台的图形界面，所以可以在所有平台上保持相同的运行效果，对跨平台支持比较出色

除此之外，Swing 提供了比 AWT 更多的图形界面组件，因此可以开发出美观的图形界面程序

特点1：轻量级组件—–采用纯 Java 实现
特点2：可插入外观组件—–允许编程人员改变外观，且外观不会随操作系统的改变而改变

Swing 包

  • java.awt.Component(图形对象):定义所有图形对象都有的方法,比如添加背景/调整大小/添加监听事件等
    • java.awt.Container(容器对象):是Component的子类,容器中可以包含其他的图形,添加了add/remove方法
      • java.awt.Window(窗口):可以展示的窗体,有如下俩子类:
        • java.awt.Dialog(对话框):是一个可弹出的对话窗口,可在其中添加组件
          • javax.swing.JDialog(对话框):sun公司在原Dialog基础上扩展出来的窗口类,是本章学东西的重点
        • java.awt.Frame(窗体):是一个窗体
          • javax.swing.JFrame(窗体):sun公司在原Frame基础上扩展出来的窗口类,是本章学东西的重点
      • javax.swing.JComponent(组件):组件时一个抽象的概念,在下面有很多子类(JPanel面板/JButton按钮/JLabel标签 等待)

一般情况下将组件放入面板中,面板放入窗口中,就会展示出我们看到的桌面程序

常用Swing组件

常用容器

  • JFrame窗体:右上角有最小化最大化关闭三个按钮
  • JDialog对话框:右上角只有关闭按钮
  • JPanel面板:没有窗体外框没有按钮,只是一个空荡荡的容器,只有放在窗体中才可以显示

其他常用组件

  • JButton按钮:可以用鼠标点击
  • JLabel标签:可以显示文字内容
  • JCheckBox多选按钮:可以一次选多条
  • JRadioButton单选按钮:一次只能选一个
  • JTextField文本框:可以输入一行文本
  • JPassword密码框:可以输入文本,会隐藏输入的内容
  • JComBox下拉框:展示选中的值,会有默认值,点击右侧三角后展示所有选项,可以选择其他值
  • JTextArea文本域:可以输入多行文本
  • JList列表框:展示所有可以选择的选项,可以单选也可以多选
  • JOptionPane小对话框:用来弹出小提示,其中可以有输入框
